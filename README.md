LegiScope

A Quizlet-style web app that helps users stay informed on legislative and political issues through active, personalized engagement.

Overview

LegiScope presents real-world legislative and political news as interactive cards. Instead of overwhelming users with endless feeds, the app surfaces one issue at a time and prompts users to engage with it by marking it as “Interested” or “Not Interested.”

This interaction model encourages consistent exposure to important civic topics while adapting to each user’s interests over time.

Features
Card-Based Learning Interface
Displays one article at a time in a clean, focused layout
Inspired by flashcard-style learning (similar to Quizlet)
Reduces information overload and promotes attention
Interest Feedback System
Users provide quick feedback on each article:
Interested → prioritizes similar topics in future cards
Not Interested → reduces similar content
Navigation Controls
Left and right arrows allow users to browse articles
Progress indicator shows position in the feed (e.g., 2 / 69)
Live Legislative Content
Articles are pulled from real news and policy sources
Covers topics such as:
Government actions
Public policy
Legislative updates
Social and economic issues
Metadata Display

Each card includes:

Headline
Source (e.g., NPR Politics)
Date
Topic tags
How It Works
The app fetches articles from external RSS feeds or APIs
Articles are stored and organized locally or in a backend system
The system presents content one card at a time
User input (Interested / Not Interested) is recorded
Future content is adjusted based on user preferences
Purpose

LegiScope is designed to improve how people engage with legislative information.

Most users either:

Ignore political content, or
Consume it passively without retention

LegiScope introduces a more effective approach:

Active interaction instead of passive scrolling
Personalized exposure based on user feedback
Incremental learning through consistent, bite-sized content
